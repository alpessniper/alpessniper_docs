## Dashboard Layout

The Alpes Sniper dashboard is designed for quick signal discovery and efficient trading decisions.

### Navigation Sidebar

The left sidebar provides quick access to all signal categories:

**Premium**
- **Premium Signals** - High-conviction curated signals

**Signals by Engine** (Age-based token discovery)
- **Token NEW** - Fresh token launches (0-48h)
- **Tokens >3D** - Tokens older than 3 days
- **Tokens >7D** - Tokens older than 7 days
- **Tokens >30D** - Established tokens (30+ days)

**Signals by Wallet**
- **Wallet Signals** - Smart Money wallet activity

**Signals by Volume**
- **Tokens High Mcap** - High market cap tokens with significant volume

**Analytics**
- **Performance** - Historical performance metrics
- **Hall of Fame** - Top performing signals

### View Modes

Switch between three display modes using the toggle in the top-right:

<CardGroup cols={3}>
  <Card title="Cards" icon="square">
    Expanded view with all details visible
  </Card>
  <Card title="Mini Cards" icon="grip">
    Compact cards, click to expand
  </Card>
  <Card title="Table" icon="table">
    Spreadsheet-style for data analysis
  </Card>
</CardGroup>

Your view preference is saved automatically and syncs across devices.

## Signal Card Anatomy

### Header Section
- **Token Logo** - Visual identifier (if available)
- **Symbol/Name** - Token ticker and full name
- **Age Badge** - Time since token creation
- **Premium Badge** - Indicates high-conviction signal (if applicable)

### Metrics Section
- **MCAP Entry** - Market cap at signal time
- **Current MCAP** - Live market cap
- **Volume 24h** - Recent trading activity
- **Holders** - Number of unique holders
- **Liquidity** - Available trading liquidity

### Status Indicators
- **DEX Paid** - Token has enhanced DEX listing
- **Socials** - Available social links (Twitter, Telegram, Website)

### Action Buttons
- **Copy Contract** - Quick copy to clipboard
- **View on Explorer** - Open in Solscan
- **Trade** - Direct DEX links

## Filtering & Sorting

### Quick Filters
Use the filter bar to narrow down signals:
- By time period (1h, 6h, 24h, 7d)
- By performance (Winners, Losers, All)
- By status (Active, Rugged, Dead)

### Sorting Options
Sort signals by:
- Newest first (default)
- Highest ROI
- Highest market cap
- Highest volume

## Real-Time Updates

The dashboard updates in real-time:
- **New signals** appear automatically at the top
- **Metrics** update periodically
- **Flash animation** highlights new arrivals

<Note>
  You don't need to refresh the page. New signals appear automatically when available.
</Note>
