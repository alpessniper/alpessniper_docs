## Overview

The **Performance** section provides detailed analytics on signal outcomes, helping you understand historical performance across different signal types.

## Key Metrics

### Win Rate

Percentage of signals that reached **2x or higher** from entry market cap.

```
Win Rate = (Winners / Total Signals) Ã— 100%
```

### Average ROI

The mean return across all signals in the selected period.

### Best ROI

The highest single return achieved by any signal.

## Filtering Options

Filter performance data by:
- **Signal Type**: All, Engine, Wallet, Premium
- **Time Period**: 24h, 7d, 30d, All Time
- **Status**: Winners, Active, Dead, All

## Understanding the Data

### Winner
Token reached 2x+ from entry market cap

### Active
Still trading, hasn't reached 2x yet

### Dead
Dropped significantly with minimal trading activity and no recovery

### Rugged
Liquidity removed or project abandoned

## Performance Chart

The chart shows:
- Cumulative performance over time
- Win rate trends
- Signal volume by category
